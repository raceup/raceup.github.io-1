Aruba

{% if site.lang == "it" %}

Il Private Cloud fornito da Aruba Cloud ci permette di disporre di una piattaforma cloud in cui è possibile compiere diverse tipologie di simulazioni, da quelle relative ai componenti aerodinamici del veicolo a quelle che riguardano le componenti strutturali della vettura che richiedono una capacità di calcolo non raggiungibile con i normali computer. A ciò si aggiunge la possibilità di raccogliere i dati telemetrici e la visualizzazione live di tali dati, fondamentali per condurre ricerche specifiche di analisi.

{% elsif site.lang == "en" %}

The Private Cloud provided by Aruba Cloud allows us to have a cloud platform in which it is possible to perform different types of simulations, from those relating to the aerodynamic components to those relating to the structural components of the car, that require a high calculation capacity that cannot be reached with normal computers. Added to this is the possibility of collecting telemetry data and the live visualization of such data, essential for conducting specific analysis research.
